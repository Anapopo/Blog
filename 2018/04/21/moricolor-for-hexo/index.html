「MoriColor」一款移植自 Typecho 的 Hexo 主题
  • hexo

最近看到一款 Typecho 主题 Moricolor ，很是喜欢，但我的博客是基于 Hexo 构建的，自然是不能通用的，于是学习了一下 Hexo 主题开发，简单地制作了这个主题样式的 Hexo 版本。

当然你现在看到的博客主题就是这个 Moricolor 了，不过目前还处于 兼容性优化(小白学习) 阶段，暂不公开 Github 项目，等到 TODO 差不多完成的时候就开源。已经开源戳我
